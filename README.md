A simple RESTful API built with FastAPI for managing users, courses, and enrollments with role-based behavior.

****
Users Can do the following

FEATURES:
1. Create user (student or admin)

2. Get all users

3. Get user by ID


VALIDATION:

1. Name cannot be empty

2. Email must be valid

3. Role must be student or admin


COURSES: The  courses feature contains tehe following

1. View all courses

2. View single course

**Admin only:**

1. Create course

2. Update course

3. Delete course

4. View all enrollments

5. Force deregister students
